# Google Merchant Center feed (TSV)

Use `data/google-merchant-feed.tsv` as your upload file for Merchant Center.

## Before uploading
1. Replace every `https://YOUR-DOMAIN.com` with your real website domain.
2. Save the file as tab-separated values (`.tsv`) in UTF-8.
3. In Merchant Center, create a **Primary feed** and choose **Upload**.

## File included
- `data/google-merchant-feed.tsv`: pre-filled with your current inventory from `data/cars.json`.
